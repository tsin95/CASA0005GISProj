Matthew Crump

His very informative guide on how to publish an R Markdown Website on Github Pages came in super handy for me here. I published this webpage with the help of files forked from his repository. Have a look to see how you can create your very own R Markdown Website too!

Emily Burchfield

Her detailed site on point pattern analysis, and especially on the G and F functions, were very enlightening for my learning and research purposes. Great analysis and very accessible content - check it out here.

Mark Burkey

His informative and step-by-step video on different spatial regression models was great! Worthy of mention was how there was a .txt file of code for the Lagrange Multiplier Tests, and how it was clearly explained. I learnt so much about the implementation of different spatial regression models from his efforts! If you’re looking for information on Spatially Lagged X models, Spatial Lag Models, or Spatial Error Models in R - here is the place.
